DataTypes in Javascript.

There are Two DataTypes in Javascript:

1. Primitive DataTypes

2. Reference DataTypes

See Figure 1.1.

Types of Primitive DataTypes:

  • Boolean
  • String
  • Numbers
  • Null
  • Undefined
  • Symbol

Types of Reference DataTypes:

  • Array
  • Dates
  • Object Literals
  • Functions

To find DataTypes of Variables we are using typeof Functions.

Syntax: (typeof Variable)

Primitive DataTypes

String:

let name = 'Saurav'; console.log('Name: '+name); console.log('DataType is '+(typeof name));
OUTPUT
Name: Saurav
DataType is string

This output will only showing the only string which is present in "console.log('Hello World');". When you try it yourself then goto your HTML page click right button on mouse and goto inspect and then goto console after that you able to see output of javascript console.
