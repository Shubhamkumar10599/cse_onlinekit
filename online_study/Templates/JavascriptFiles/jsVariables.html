Variables in Javascript. Let,Const,Var

Rules for Javascript Variables:

(a) Javascript Variables cannot start with number.

(b) Javascript Variables can start with underscore(_),dollar($) and Letter.

(c) Javascript Variables are cases sensitive.

(d) Javascript Variables cannot contain spaces between them like (vari ables) wrong here.

Example like if,else,for,while,do etc are certain inbuilt keywords which are not used as variables.

WE highly recommended to not use dollar($) and underscore(_) as starting character of variables because it is mainly used as keywords in advanced javaScript framework like JQUERY.
