C language is considered as the mother language of all the modern programming languages.

What is C ?

C is a programming language developed at AT & T's Bell Laboratories of USA in 1972.

It has design and written by a man named Dennis Ritchie.

C became popular because it is reliable, simple and easy to use.

Why to learn C first ?

C has been already superseded by Language like C++, C#, and Java, so why bother to learn C today.

C++, C#, or Java make use principal called "object oriented programming" to organize the program. This organizing principal has lots of advantage to offer. But even while using this organizing principal, you still need a good hold over the language elements of C and the basic programming skills.

Major parts of popular operating Systems like Windows, UNIX, Linux and Android are written in C.

I hope that these are very convincing reasons why would adopt C as the first.
