Types of C Variables

(b) Real Variables: It can hold only an Real constant.

(a) Character Variables: It can hold only an Character constant.

Rules for Constructing Variables name

(a) A variables name is any combination of 1 to 31 alphabets, digits or underscores.

(c) Some compilers allow variable whose names length could be up to 247 characters.

(d) But this is not a good practice to create unnecessary long variables names.

(e) Variable names must be meaningful according to program.

(f) The first character in the variable name must be an alphabet or underscore( _ ).

(g) No commas or blanks are allowed within the variable name.

(h) No special symbol other than an underscore (as in gross_sal) can be used in a variable name.

While Learning C, We first learn alphabets, digits and special symbols.

Then we learn how to combine them to form constants, variables and keywords.

And finally, how to combine them to form an instruction.

A group of instructions can then be combined to form a program.
